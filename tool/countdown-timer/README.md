# 倒計時計時器 (Countdown Timer)

一個現代化、深色主題的倒計時計時器，使用純 JavaScript、CSS 和輕量級 library 製作。

## 功能特性

- ✅ **計時器控制**: 開始、暫停、繼續、停止、重置
- ✅ **實時顯示**: HH:MM:SS 格式，防止時間漂移
- ✅ **狀態指示**: 顯示準備就緒、運行中、已暫停、已完成
- ✅ **預設按鈕**: 快速選擇 1、5、10、15、25、30、45、60 分鐘
- ✅ **自訂輸入**: 輸入自訂時、分、秒，包含驗證
- ✅ **通知**: 計時完成時顯示 Toast 通知和聲音提醒
- ✅ **深色主題**: 現代漸層背景，藍色強調色
- ✅ **響應式設計**: 支援桌面和行動設備
- ✅ **鍵盤快捷鍵**: 空格鍵暫停/繼續，Esc 停止
- ✅ **無障礙**: ARIA 標籤、語義化 HTML

## 技術棧

- **純 JavaScript** - 無框架依賴
- **CSS3** - 動畫和響應式設計
- **Font Awesome** - 圖標
- **Web Audio API** - 聲音通知

## 使用方法

1. 直接在瀏覽器中打開 `index.html`
2. 選擇預設時間或輸入自訂時間
3. 點擊播放按鈕開始倒計時
4. 使用暫停/繼續控制計時器
5. 計時完成時會收到通知

## 快捷鍵

- **空格鍵**: 暫停/繼續
- **Esc**: 停止計時

## 文件結構

```
countdown-timer/

├── styles.css      # 樣式和動畫
├── timer.js        # 計時器邏輯

```

## 特點

- **無依賴**: 只需要 Font Awesome CDN，無需構建工具
- **輕量級**: 總大小不到 50KB
- **易於集成**: 可直接複製到任何網站
- **防止時間漂移**: 使用 `Date.now()` 而非依賴 interval 計時
- **完全響應式**: 在所有設備上完美運作
